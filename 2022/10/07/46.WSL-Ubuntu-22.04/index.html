WSL 升级 Ubuntu 22.04
发布于：2022-10-07 15:09:54 标签：/ WSL / 访问：

看到 Ubuntu 22.04 释出，尝试升级 WSL，执行 sudo do-release-upgrade 提示有个包无法获取：

这个问题一般是网络原因，办法之一是换国内源。这里说明下，我们需要修改源为 22.04 的 jammy，而不是 20.04 的 focal，为什么呢？因为大版本更新是从 jammy 获取，如果修改为 focal 还是会出现上面的问题。

1. 备份文件

执行 sudo cp /etc/apt/sources.list /etc/apt/sources.list.backup 备份文件，以防意外。

2. 修改文件

执行 sudo vim /etc/apt/sources.list，按 Insert 键修改内容为国内源，然后按 Esc 键输入 :wq 保存。

3. 更新源

执行 sudo apt update 和 sudo apt upgrade，等待更新完成，执行 lsb_release -dc 显示 Ubuntu 22.04 代表更新成功。
