Hexo博客搭建教程
发布于：2017-01-08 18:00:00 标签：/ Hexo / 访问：

在 Github Pages 搭建基于 Hexo 框架的博客，快速上手而且简洁高效，详见 Hexo 文档。

2022-03-15

因为设备换为 Mac，所以更新下教程。
macOS 使用 Homebrew 安装 Node.js，所以先用以下命令安装 Homebrew：
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
然后安装 Node.js：
brew install node
然后安装 Git：
brew install git
然后跳到下面第 5 步安装 Hexo，后续步骤一致。

1. Node.js

下载链接：32bit&64bit，不确定版本的话直接下载32位即可，然后保持默认设置安装。
检查安装情况，win+r 打开运行，输入 cmd 回车，输入 node -v 和 npm -v，如下图所示，则说明安装正确。

2. Git

下载链接：32bit&64bit，其他设置只需要保持默认，但建议 PATH 选项按照下图选择：

检查安装情况，输入 git --version，如下图所示，则说明安装正确。

3. Github

打开 https://github.com/ 注册帐号，前往邮箱确认注册。登录帐号，点击页面右上角的+，选择 New repository，注意在 Repository name 下必须填写为 yourname.github.io，其他可不填，然后点击 Create repository。

4. GitHub Pages

可以去看这里的官方教程，只要 yourname.github.io 可以正常访问就可以。

5. Hexo

任意盘符下新建文件夹，打开文件夹右键选择 Git Bash Here，如图：

输入 npm install hexo-cli -g 看到下图：

继续输入 npm install hexo --save，结束后输入 hexo -v，看到下图就说明安装成功。

输入 hexo init 初始化，然后输入 npm install 等待 npm 安装完成后，再次输入 hexo g 然后 hexo s，这时在浏览器中打开 http://localhost:4000/，就看见了你的博客首页。
